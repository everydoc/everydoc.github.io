PowerDesigner 使用记录

1. 连接Oracle

选择的jar包一定要配置在CLASSPATH环境变量中，否则会出现连接不上的问题。

2. 导出代码
   1. 反向工程生成PhysicalDataModel，
   2. Ctrl+Shift+O 生长ObjectOrientedModel
   3. 选择Language菜单下的Generate Java Code

3. 生成测试数据
